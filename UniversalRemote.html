1. General Information

Many script calls and plugin commands accept either multiple inputs, or arrays as inputs.

If multiple inputs are accepted, they will be denoted with a ^. If array inputs are accepted, they are denoted with a *. Some functions accept both multiple and array inputs.

  • Script Call: this.SearchNames(names^*, numberV, arrayV, condition)

What this means: In the names field, you can enter

  • "Tom"
  • "Tom", "Dick", "Harry"
  • ["Tom", "Dick", "Harry"]
  • ["Tom", "Dick"], "Harry"
  • ["Tom", "Dick", "Harry"], ["Susan", "Mary", "Anne"]

If you use the plugin command for the same function, on the other hand, you cannot use arrays.

  • Plugin Command: SearchNames names^ numberV arrayV condition

What this means: In the names field, you can enter

  • Tom
  • Tom Dick Harry

Search Engine

Live Search Engine: Search in-game for groups of Event IDs by setting conditions, using their names, notes and even self-switches! Do searches even in conditional branches!
